List Tree View

This alternative view mode for the control flow tree is based on a more compact list representation. In comparison to the default visualization, it puts less emphasis on the search tree by only displaying one branch at a time, but allows the user to switch between alternative branches at each choice point. The resulting less hierarchical structure uses up a lot less screen space, leaving more real estate for other views, or allowing more steps to be displayed at the same time. On the other hand, the global structure of the search tree is not visible any longer, which is a disadvantage if the user wants to find the branches where most steps occurred.
